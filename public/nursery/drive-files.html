System Configuration Required

To connect the Real Google Drive, you need to provide API credentials. This is a security requirement from Google.

Instructions for IT Admin:
  1. Go to Google Cloud Console and create a project.
  2. Enable the Google Drive API.
  3. Create an API Key and OAuth Client ID.
  4. Open scripts/config.js in the project folder and paste them there.
